## SQL Policy Example

### Before (Broken)
```sql
-- Only authenticated users can read
CREATE POLICY "products_auth_select" 
ON public.products
FOR SELECT 
TO authenticated  -- âŒ Too restrictive
USING (is_active = true);
```

### After (Fixed)
```sql
-- Anonymous + authenticated can read
CREATE POLICY "products_public_select" 
ON public.products
FOR SELECT 
TO anon, authenticated  -- âœ… Allows public access
USING (is_active = true);

-- Admins can still do everything
CREATE POLICY "products_admin_all" 
ON public.products
FOR ALL 
TO authenticated
USING (
  EXISTS (
    SELECT 1 FROM users 
    WHERE id = auth.uid() 
    AND is_admin = true
  )
);
```
